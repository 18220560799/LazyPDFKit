# LazyPDFKit
An IOS PDF Framework written in Objective-C

![Screenshot1](/../master/Screenshots/Screenshot1.png?raw=true "Screenshot1")

#Features
* PDF annotation
* Free hand draw tool
* Insert text
* Highlight text
* Draw line
* Draw boxes and circle
* Undo, Redo and Clear
* Change Color, Size and Opacity
* Thumbanail View
* Print
* Bookmark
* Email PDF
